Sample Navigation System for The Compass Frameworks

Sample Nav Menu for the Frameworks. Steps 1 - 3 show steps that have been successfully completed, other steps are yet to be started or are not yet finished. Search function at the top of the navigation. The specific client name is also highlighted as a sub heading in the nav menu. There will potentially be multiple clients going through the process at any one time, so we'll need to determine how to "launch" into this menu system and pass along a "client org" at some point in the future.
